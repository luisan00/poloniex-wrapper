# poloniex-wrapper
Wrapper for Poloniex API methods .

## What is this:
Simple wrapper in python for poloniex trading platform.

## Requeriments
- Python 3+
- autobahn
## How to
Install dependencies with pip:

> pip install autobahn

or with pip3:

> pip3 install autobahn

Copy/Download or clone to your project.

> git clone
